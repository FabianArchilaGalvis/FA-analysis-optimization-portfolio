# FA-analysis-optimization-portfolio
- Fabian Archila, ingeniero industrial UIS con experiencia en análisis y optimización en las diferentes áreas de una empresa.
- Fabian Archila, UIS industrial engineer with experience in analysis and optimization in different areas of a company (English below the spanish text).

Este repositorio presenta un caso práctico donde se transforman datos operativos en información útil para apoyar la toma de decisiones, con un enfoque generalista y aplicable a distintas áreas de la empresa.

## Contenido del repositorio
- `supplier_performance_analysis/`  
  Proyecto de análisis de desempeño de proveedores:
  - `data/` → datos base (CSV)
  - `outputs/` → resultados del análisis (Power BI, Excel, PDFs)
  - `inbound_analysis.py` → análisis reproducible en Python
  - `README.md` → descripción específica del proyecto
---
## Habilidades utilizadas
- Limpieza y preparación de datos reales.
- Definición de KPIs relevantes para evaluación comparativa.
- Análisis de trade-offs (costo vs tiempo).
- Normalización y construcción de scores para priorización.
- Visualización y comunicación de resultados orientados a decisión.
- Enfoque de ingeniería para identificar oportunidades de mejora y optimización.
-  **Nota**: el análisis presentado es un ejemplo de aplicación. La metodología utilizada puede adaptarse a múltiples contextos organizacionales (operaciones, logística, compras, finanzas, calidad, etc.), dependiendo de la disponibilidad y naturaleza de los datos.
